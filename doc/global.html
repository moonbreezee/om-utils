debounce(func, wait, immediate) → {function}

与throttle不同的是，debounce保证一个函数在多少毫秒内不再被触发，只会执行一次，要么在第一次调用return的防抖函数时执行，要么在延迟指定毫秒后调用。

事件停止被触发N秒后才会再次触发回调，防抖动函数

回调执行函数

是否延时执行

once(fn, context)

实用的执行一次函数，不用多解释。虽然很简单的函数，但是防止重复加载或者初始化的习惯必须养成。

once

Parameters:
Name Type Description
fn function

需要执行一次的函数，被包装后，只有第一次会执行
context *

当前函数执行的上下问环境
Source:
  • advancedFunction/once.js, line 20
Example
var canOnlyFireOnce = once(function () {
  console.log('Fired!');
});
canOnlyFireOnce(); // Fired
canOnlyFireOnce();
